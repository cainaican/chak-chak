## Промпт для таплинка
Сделай одностраничный (single-page) сайт - taplink в формате link-in-bio (как Linktree) для проекта «Чак-Чак Дискотека» и его организатора Булата Миндиярова.

Требования:
 1. Общий стиль:

 • Дизайн в татаро-башкирской эстетике
 • Используй национальные орнаменты, узоры, тёплые цвета (золото, бордовый, тёмно-зелёный)
 • Современный минимализм + этнические элементы
 • Атмосфера: весёлая, праздничная, немного ностальгическая (дискотека + национальный колорит)

 2. Структура страницы:

 • Аватар / логотип (заглушка)
 • Название: «Чак-Чак Дискотека»
 • Подзаголовок: «Татар-башкирский вайб, музыка и тусовки»
 • Блок с именем: «Булат Миндияров — организатор»

 3. Блок ссылок (кнопки):
Сделай красивые кнопки с hover-эффектами для:

 • Instagram (Чак-Чак Дискотека)
 • Instagram (Булат Минтияров)
 • Telegram канал
 • Telegram чат
 • TikTok
 • YouTube
 • Сайт / афиша мероприятий 
 • Кнопка «Купить билет»

(ссылки можно пока оставить как #)
 4. Функциональность:

 • Адаптивность (mobile-first)
 • Плавные анимации при наведении
 • Лёгкий градиентный фон или декоративный паттерн
 • Можно добавить лёгкую анимацию (например, мерцание или движение орнамента)

 5. Технологии:

 • Чистый HTML + CSS + немного JavaScript (без фреймворков)
 • Весь код в одном файле (index.html)

 6. Дополнительно:

 • Добавь немного татарских/башкирских слов (например: «Рәхим итегез», «Дуслар!»), но не перегружай
 • Сделай шрифт читаемым, но с намёком на этнический стиль
 • Добавь иконки соцсетей (можно через CDN)

Сгенерируй полностью готовый код страницы, и сделай таким чтобы можно было в нем разобраться чтобы делать быстрые правки
